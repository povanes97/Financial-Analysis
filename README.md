# Public Mutual Fund Performance Analysis

## Project Overview
This project analyzes Public Mutual funds across multiple investment horizons:
- YTD
- 1 Year
- 3 Year
- 5 Year
- 10 Year

## Objectives
- Identify top-performing funds
- Evaluate consistency across investment horizons
- Assess the trade-off between liquidity and long-term growth

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Methodology
1. Data cleaning
2. Return analysis
3. Top 50 ranking
4. Consistency score calculation
5. Growth vs Flexibility scatter plot

## Key Findings
- Long-term winners are not always short-term winners.
- Consistency Score identifies funds performing well across multiple horizons.
- Several funds provide both strong long-term growth and competitive short-term returns.
